# 🚕 NYC Taxi Fare Predictor (ML Fullstack App)

An end-to-end Machine Learning web application that predicts New York City taxi fares based on TLC (Taxi & Limousine Commission) data. Built with **LightGBM**, **FastAPI**, **React**, and fully containerized with **Docker**.


## ✨ Features
* **Machine Learning Engine:** Trained on millions of NYC Taxi records using LightGBM.
* **Confidence Intervals:** Uses Quantile Regression to provide a realistic expected price range (Min/Max).
* **Enterprise API:** Fast, asynchronous backend built with Python's FastAPI.
* **Modern UI:** Responsive, enterprise-grade React frontend.
* **One-Click Setup:** Fully containerized with Docker Compose for seamless deployment across any OS.

## 🛠️ Tech Stack
* **Data Science / ML:** Pandas, Scikit-Learn, LightGBM, MLflow
* **Backend:** Python, FastAPI, Uvicorn, Pydantic
* **Frontend:** React (Vite), Axios, Lucide-Icons, CSS3
* **DevOps:** Docker, Docker Compose
